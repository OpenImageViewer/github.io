Open Image Viewer Feature Highlights

Feature Description Tutorial
Quick paste Double right mouse click to paste an image from the clipboard
Optimized selection rectangle High performance and visually optimized selection rectangle to crop or copy part of the image Video Tutorial
persistence view keep zoom and pan unchanged when loading a new file.
Useful when having similar sized images and looking for how pixels change in a specific region of an image or when knowing how the image is supposed to look likebut the user is uncertain
Infinite panning Low level input techniques allow to “move” the mouse far beyond the edges of the monitor
Multi size image support Support for multi image format such as DDS mipmaps, multi-size icon and multi-image Tiff, etc.
Deferred initialization Much of the startup execution is deferred after the initial image has been displayed
Hardware color correction Gamma, exposure, Offset and Saturation are executed on the GPU
Downscaling Improve the quality for large images when zooming out using an asynchronous box filter to eliminate aliasing
On Screen display Notification message for every operation is rendered directly into the image
32 bit clipboard support Copy and paste to/from the clipboard with a transparency channel. It may seem trivial but many renowned applications don't support it
Checkerboard background Image background is a black/blue checkerboard grid. This helps identify the edges of an image in case the image edge color is the same as the background color, if it’s solid
Grayscale / rainbow normalization Images that cannot be directly converted to the RGB (the monitor format) usually have to be normalized either to grayscale (black-to-white) or either to colorful image (red to magenta, via the primary colors green and blue) Video Tutorial
Multiple transparency modes Four transparency checkerboard visualization modes: light, medium, dark and darker. Helps to clearly visualize transparency for images with different luminosity Video Tutorial
Border-less mode Completely remove the window frame for utilizing more screen space
Easy drag Drag the window from anywhere, not just from the title
Easy resize resize the window from anywhere not just from the borders by holding the CTRL key
Pixel grid
On-the-fly Texel value Display the storage texel value (the value of the original image before rasterization) where the mouse cursor is positioned while preserving the order of the color/data channels
ZoomSense Sensing how much zoom to apply when the user interacting with the mouse wheel
Texel / sub-texel position Displays the sub-texel position of the image where the mouse is positioned.
Multi-full-screen allows to span an image across several monitors even with different resolutions Video Tutorial
Auto on top keep the window on top for three seconds so it won’t hide behind other windows. Useful for drag and drop
Conservative axis aligned transformations Axis aligned transformations such as rotation and flip are resolved according to the initial transformation state while keeping rotation as the primary operation.
For example, Flipping an image horizontally and then rotating it 90 degrees clockwise will be resolved and reported to the user as a transformation that is rotated 90 degrees clockwise and flipped vertically
C API viewing engine Open image viewer is also a C API image viewing engine, use it in your application
