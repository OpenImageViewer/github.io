Designed for ergonomics

  • Use of low level input API to minimize mouse movements
  • Zoom is adaptive to minimize mouse wheel finger gestures
  • Four transparency modes to minimize eyes strain
  • Dynamic virtual canvas limits for placing any texel on any position on the screen
  • Pixel grid for easily identifying distinct pixels
  • Two color checkered background to identify the edges of the images
  • No delay or animation when displaying the image for immediate response as other viewers do
  • Configurable key bindings using an external JSON keybinidngs descriptor

Designed for performace

  • Built for real-time graphics like a modern game
  • Fully hardware accelerated by using Low level graphics API such as Direct3D
  • Custom shaders for every genereated pixel
  • Built from scratch using modern C++20 techniques
  • Safe and stable
  • Lightweight with minimum external dependencies
  • Profiling and optimized is a daily routine
  • Uses specialized codecs for fastest implementations
