About
Open Image Viewer is a hardware accelerated, light-weight and stand-alone open code image viewer.
A prominent guideline of Open Image Viewer is attributed to Human factors and ergonomics by optimizing input (mouse / keyboard) and output (monitor) to the human physiology, thus doing more with less.
The Viewer aims to be the status-quo in viewing and analyzing images with no compromise in performace, comfort or universality. The project is built from scratch using modern techniques, modern libraries and non legacy constraints.
